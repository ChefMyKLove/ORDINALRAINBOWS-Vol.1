Welcome to the official sales and tracking page for the ORDINAL RAINBOWS Vol. 1 NFT collection!

## Project Overview
This project showcases a visually stunning NFT gallery featuring the ORDINAL RAINBOWS Vol. 1 collection. Each NFT card displays unique artwork, rarity, and details, with a glassmorphism-inspired design and animated rainbow background. The site also includes a modal for ordering physical prints via Printify.

## Features
- **NFT Gallery:** Responsive grid of NFT cards with image, title, subtitle, rarity, and details.
- **Glassmorphism Design:** Modern, frosted-glass effect for cards and overlays.
- **Animated Rainbow Background:** Fullscreen background cycles through collection images for a vibrant effect.
- **Corner Shine Effect:** Subtle edge highlights on NFT cards to make them pop.
- **Printify Integration:** Modal window for ordering prints of your favorite NFTs.
- **Mobile Friendly:** Layout adapts for smaller screens.

## Getting Started
### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)
- [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) (optional, for advanced development)
- [Git](https://git-scm.com/) (for cloning and contributing)

### Installation
1. **Clone the repository:**
   ```sh
   git clone https://github.com/ChefMyKLove/ORDINALRAINBOWS-sales-and-tracking.git
   ```
2. **Navigate to the project folder:**
   ```sh
   cd ORDINALRAINBOWS-sales-and-tracking
   ```
3. **Open `RainbowInfo.html` in your browser.**

## Usage
- Browse the NFT gallery to view all items in the collection.
- Click "Order Prints" on any card to open the Printify modal and purchase physical prints.
- Enjoy the animated rainbow background and glassmorphism effects.

## Customization
- **Add new NFTs:** Edit the `nfts` array in `RainbowInfo.html` and add corresponding images to the `images/` folder.
- **Change styles:** Modify CSS in `RainbowInfo.html` or the `styles/` folder.
- **Update Printify links:** Adjust modal logic in the script section if your Printify store changes.
